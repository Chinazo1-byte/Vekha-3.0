# Веха

Десктопное приложение для педагогов-психологов, дефектологов и логопедов. Позволяет создавать коррекционные упражнения, собирать занятия из цепочек и проводить психологические диагностики — всё в офлайн-режиме, без регистрации и облака.

Работает на Windows и Linux (включая Astra Linux).

---

## Возможности

**Упражнения** — 18 типов, охватывающих основные направления коррекционной работы:

| Группа | Типы |
|---|---|
| Восприятие и сопоставление | Сопоставление, Мемо, Найди пару, Лишний предмет |
| Память и внимание | Что исчезло? |
| Мышление и классификация | Сортировка, Три группы, Последовательность, Продолжи ряд, История по порядку |
| Речь и грамота | Слово → картинка, Составь слово, Вставь слово, Первый звук |
| Математика | Считаем, По размеру, Сравни |
| Эмоции | Верно / Неверно, Назови эмоцию |

Каждый тип имеет визуальный редактор и интерактивный плеер с поддержкой drag-and-drop и тапов. Результат каждого упражнения фиксируется автоматически.

**Занятия** — цепочки из любого количества упражнений. Можно задать порядок, запустить последовательно и получить сводный результат по занятию.

**Ученики** — карточка ученика с историей занятий, динамикой результатов и прикреплёнными диагностиками.

**Диагностики** — встроенная библиотека стандартизированных методик и конструктор собственных. Поддерживаются вопросники с нормативными шкалами, автоматический подсчёт баллов и интерпретация результатов. Можно импортировать методику из JSON-файла.

**Экспорт** — PDF-отчёты по результатам диагностик и занятий. Библиотека упражнений экспортируется и импортируется через ZIP-архив с вложенными изображениями.

## Требования

- Node.js 18 или новее
- Windows 10/11 или Linux (x64)

На Astra Linux дополнительно нужен пакет `libfuse2` для запуска AppImage.

---

## Запуск из исходников

```bash
# Установить зависимости
npm install

# Запуск
npm start

# Запуск с DevTools
npm run dev
```

---

## Сборка дистрибутива

```bash
# Windows — NSIS-установщик
npm run build

# Linux — AppImage
npm run build:linux

# Оба сразу
npm run build:all
```

Артефакты сборки появятся в папке `dist/`.

## База данных

SQLite через sql.js (in-memory, сохраняется на диск через IPC). Файл базы:

- Windows: `%APPDATA%\vekha\vekha.db`
- Linux: `~/.config/vekha/vekha.db`

При обновлении приложения данные сохраняются. Миграции схемы выполняются автоматически при каждом запуске.

## Формат JSON для диагностик

Методику можно подготовить заранее и импортировать через интерфейс.

```json
{
  "name": "Название методики",
  "description": "Краткое описание",
  "fill_by": "teacher",
  "questions": [
    {
      "id": "q1",
      "text": "Текст вопроса",
      "type": "likert",
      "options": ["Никогда", "Иногда", "Часто", "Всегда"]
    }
  ],
  "scales": [
    {
      "id": "total",
      "name": "Общий балл",
      "questions": ["q1"],
      "norms": [
        { "min": 0, "max": 10, "label": "Норма", "color": "green" },
        { "min": 11, "max": 20, "label": "Риск", "color": "amber" }
      ]
    }
  ]
}
```

Поддерживаемые типы вопросов: `likert`, `choice`, `yesno`, `number`, `text`.
