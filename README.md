# Ясная Грань — Electron

Десктопное приложение для педагогов: конструктор упражнений, занятий и диагностик.

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Запустить
npm start

# Для разработки (открывает DevTools)
npm run dev
```

## Требования
- Node.js 18+
- Windows / macOS / Linux

## Структура

```
main.js          — Electron main process, БД, IPC-обработчики
preload.js       — Безопасный мост renderer ↔ main
src/
  index.html     — Оболочка приложения
  styles/
    main.css     — Дизайн-система (токены, компоненты)
  js/
    utils.js     — Общие хелперы, Modal, Toast, Icons
    router.js    — Переключение страниц
    app.js       — Инициализация
    pages/
      students.js     — Ученики
      exercises.js    — Упражнения
      sessions.js     — Занятия (цепочки)
      diagnostics.js  — Диагностики
```

## База данных

SQLite, файл хранится в `userData` Electron:
- Windows: `%APPDATA%/yasnaya-gran/yasnaya-gran.db`
- macOS: `~/Library/Application Support/yasnaya-gran/yasnaya-gran.db`

## Формат JSON для диагностик

```json
{
  "name": "Название методики",
  "description": "Описание",
  "fill_by": "teacher",
  "questions": [
    {
      "id": "q1",
      "text": "Текст вопроса",
      "type": "likert",
      "options": ["Никогда", "Иногда", "Часто", "Всегда"]
    }
  ]
}
```

## Что реализовано в сессии 1

- Дизайн-система (шрифты Onest + Playfair Display, тёплая палитра)
- Навигация с 4 разделами
- Ученики: список, добавление, редактирование, удаление, боковая панель с историей
- Упражнения: сетка, фильтры по типу/сложности, создание, удаление
- Занятия: цепочки упражнений, шаблоны, конструктор порядка
- Диагностики: список, загрузка JSON, создание вручную
- Все данные в SQLite офлайн
