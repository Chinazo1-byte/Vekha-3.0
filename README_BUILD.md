# 🔧 Инструкция по сборке Вехи в .exe

## Что понадобится

- **Node.js** v18 или выше → https://nodejs.org (скачать LTS)
- **Windows 10/11** (сборку под Windows лучше делать на Windows)
- Распакованная папка `vekha-v1` из архива

---

## Шаг 1 — Установить зависимости

Откройте папку `vekha-v1` в терминале (PowerShell или cmd):

```
cd vekha-v1
npm install
```

> Первый раз займёт 2–5 минут — скачивается Electron (~120 МБ) и electron-builder.

---

## Шаг 2 — Проверить, что приложение запускается

```
npm start
```

Должно открыться окно Вехи. Если всё работает — закройте и переходите к сборке.

Для разработки (с DevTools):
```
npm run dev
```

---

## Шаг 3 — Собрать установщик .exe

```
npm run build
```

После завершения (3–10 минут) в папке `vekha-v1/dist/` появится:

```
dist/
  Веха Setup 1.0.0.exe   ← установщик для пользователей
  win-unpacked/          ← готовая папка (без установки)
```

---

## Шаг 4 — Распространить

Передайте педагогам файл **`Веха Setup 1.0.0.exe`**.

При установке:
- Программа появится в меню Пуск как «Веха»
- Создастся ярлык на рабочем столе
- Данные хранятся в `%APPDATA%\vekha` (не удаляются при переустановке)

---

## Иконка приложения

Для красивой иконки в .exe нужен файл **icon.ico** (256×256 px).

Конвертировать logo.jpg → icon.ico можно на сайте: https://convertio.co/jpg-ico/

После конвертации положите `icon.ico` в папку `src/assets/` и обновите `package.json`:
```json
"win": {
  "icon": "src/assets/icon.ico"
}
```

## Возможные проблемы

### `npm install` зависает или ошибка сети
- Попробуйте: `npm install --registry https://registry.npmjs.org`

### Ошибка `NSIS` при сборке
- Убедитесь, что антивирус не блокирует `electron-builder`
- Запустите PowerShell от имени администратора

### Видео не воспроизводится в собранном .exe
- Убедитесь, что `src/assets/splash.mp4` включён в сборку (он уже прописан в `"files"` в package.json)

### Приложение открывается не в полноэкранном режиме
- Это нормально при запуске через `npm start` (dev-режим). В собранном .exe — полный экран.
- При отладке используйте `npm run dev`

## Структура файлов

```
vekha-v1/
├── main.js              — главный процесс Electron
├── preload.js           — мост между renderer и main
├── package.json         — конфигурация проекта и сборки
├── report_template.js   — шаблон PDF-отчётов
└── src/
    ├── index.html       — главная страница
    ├── assets/
    │   ├── logo.jpg     — логотип
    │   ├── splash.mp4   — заставка при запуске
    │   └── sounds/      — звуковые эффекты (10 файлов)
    ├── styles/
    │   └── main.css
    └── js/
        ├── app.js       — инициализация
        ├── sound.js     — звуковой движок
        ├── player.js    — плеер упражнений
        ├── session_player.js
        ├── editor.js
        └── pages/       — страницы (ученики, упражнения, занятия, диагностики)
```

## Обновление приложения

Чтобы выпустить новую версию:
1. Измените `"version"` в `package.json` (например, `"1.1.0"`)
2. Запустите `npm run build`
3. Передайте новый `.exe` педагогам — при установке поверх старой версии данные сохранятся
